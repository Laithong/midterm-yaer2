  1. code web ເປັນພາສາທີໃຊ້ສຳລັບຂຽນເວັບຄື
    HTML ຍໍ້ມາຈາກ Hyper Text Markup Language ເປັນພາສາຂຽນເວັບ ໃຫ້ໜ້າເວັບສະແດງ ຊີ່ງເປັນພາສາຫຼັງທີ່ຕ້ອງໃຊ້ຂຽນເວັບ
    CSS ຍໍ້ມາຈາກ Cascading Style Sheets ແມ່ນພາສາທີ່ຈັດຮູບແບບຂອງເອກະສານ HTML ໃຫ້ມີຄວາມງາມຫຼາຍຂື້ນ ເຮັດໃຫ້ເວັບພັດທະນາໄວຂື້ນ ແລະ ມັນສາມາດຄວບຄຸມຮູບແບບຂອງຫນ້າເວັບຕ່າງໆໃນເວລາດຽວກັນ
    JavaScript ແມ່ນພາສາທີ່ສາມາດປ່ຽນແປງເນື້ອຫາ ສາມາດປ່ຽນແປງຄ່າ ຄຸນລັກສະນະ ສາມາດຊ່ອນຫົວຂໍ້ HTML ໄດ້, ຊື່ງເປັນເຄື່ອງມືທີ່ໃຊ້ສຳລຳຂຽນເວັບ.
    ດັງນັ້ນ ຄົນທີ່ຕ້ອງການມີເວັບໄຊທ໌ເປັນຂອງຈົວເອງ ຕ້ອງໄດ້ສືກສາທັ້ງສາມພາສານີ້ ເພື່ອໃຫ້ເວັບໃຊ້ຂ້ອງທ່ານນັ້ນສົມບູນແລະສວຍງາມ
    ແຕ່ ໃນປັດຈຸບັນການຂຽນເວັບໄຊທ໌ ເປັນເລື່ອງທີງ່າຍ ເພາະວ່າມີແມ່ແບບມາໃຫ້ເບີ່ງ ບໍ່ວ່າຈະເປັນ: https://www.w3schools.com ແລະ https://getbootstrap.com ພຽງແຕ່ທ່ານເອົາໄປປະຍຸກໃຊ້ໃຫ້ເປັນ.
